my main development project is graphene (likely down for the time being, you can view the source code on github). ive also made a few fun web pages:

  • this poisson graph is an interactive poisson graph, this is very useful for visualizing how the mean affects the shape of the distribution.
  • gravity is a 2d newtonian gravity simulator, you can add an object by clicking, hold down to add mass, dragging for velocity, n letting go to release.
  • revolution.js is a normally distributed cluster of (h&s) sprites with a simulated spherical wave expanding outward, this is built using three.js.
  • harp.js is a harp created by iteratively interpolating two bezier curves, it doesnt work well on mobile ): this uses traer.js for physics handling n midi.js for the notes.

there are also a bunch of javascript libraries ive made during the development of graphene at scripts.phene.co.
